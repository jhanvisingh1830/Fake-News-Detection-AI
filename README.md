# Fake News Detection AI

A Machine Learning + NLP web application that detects whether news is fake or real.

## Technologies Used

- Python
- Flask
- Scikit-learn
- NLP
- TF-IDF
- HTML/CSS

## Features

- Fake news prediction
- Confidence score
- Modern web interface
- Machine Learning model integration

## How It Works

1. User enters news text
2. Text preprocessing is applied
3. TF-IDF converts text into vectors
4. ML model predicts fake or real news
5. Result displayed on website
